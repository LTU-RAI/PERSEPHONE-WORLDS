# PERSEPHONE-WORLDS

![alt text](assets/persephone_blue_2.webp)

## Overview 

A collection of world environments from collected and modeled information within the PERSEPHONE project. The mesh quality can be improved by using BLENDER https://www.blender.org/ to remesh to desired level of detail.

This project has received funding from the European Union’s Horizon Europe Research and Innovation Programme under the Grant Agreement No.101138451

More details about the project can be acceessed here: https://www.persephone-mining.eu
